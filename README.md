# Smart Privacy Mask (智能隐私盾)

基于端侧 AI 技术的微信小程序脱敏系统，保护您的个人隐私信息。

## ✨ 特性
- **全本地处理**：基于 ONNX Runtime，所有识别与打码均在手机端完成，隐私零上传。
- **情境感知**：自动识别姓名、身份证号、手机号等实体标签，实现差异化脱敏。
- **批量处理**：支持最多 9 张图片同时上传并一键自动化打码。
- **手动补漏**：支持手动涂抹模式，确保 100% 安全。

## 🛠️ 技术栈
- 微信小程序原生框架
- PaddleOCR v3 (DET + REC)
- ONNX Runtime Web (WASM 推理加速)

## 📖 如何使用
1. 下载本项目代码。
2. 使用微信开发者工具打开。
